# pyflipt
Python asyncio client for Flipt rest api

[Flipt REST API documentation](https://flipt.io/docs/api/)


## Supports

 - [x] Creating and deleting objects: Flags, Segments, Constraints, Rules
 - [ ] Support for Distributions and Variants
 - [ ] Updating objects
 - [ ] Listing
 - [ ] Evaluate
 - [x] Order rules
 - [ ] Getting objects
